# Venue Booking Management System

## Overview

The **Venue Booking Management System** is a software solution designed to simplify the process of managing venue reservations. It allows users to view available venues, make bookings, and manage reservations efficiently, while providing administrators with control over scheduling and data management.

---

## &#x20;Features

* Venue listing and availability checking
* Booking and reservation management
* User-friendly interface for smooth interaction
* Conflict-free scheduling system
* Admin controls for managing venues and bookings
* Data storage and retrieval

---

## Technologies Used

* Programming Language: *Java*
* Database: *SQL Server*
* Framework/Libraries: *JavaFX*
* Tools:* Git, Github, Ecllipse IDE, IntelliJ IDEA*

## &#x20;Installation

1. Clone the repository:

```
git clone https://github.com/your-username/venue-booking-system.git
```

2. Navigate to the project directory:

```
cd venue-booking-system
```

3. Compile/Run the project:

```
# Run the application
java --module-path /path/to/javafx/lib --add-modules javafx.controls,javafx.fxml -jar VenueBookingSystem.jar
```

---

## Usage

* Launch the application
* Browse available venues
* Select desired date and time
* Make a booking
* Admins can log in to manage venues and reservations
